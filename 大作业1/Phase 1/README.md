# 基于 MongoDB 的全功能动态问卷系统 (Survey System)

华东师范大学 2026《数据管理系统》课程 Project 1 第一阶段作业

## 🌟 项目简介
本项目是一个极简但功能完备的在线问卷系统。采用前后端分离的思想，后端基于 Python Flask，数据库采用 MongoDB，充分发挥了 NoSQL 文档型数据库在**处理动态不规则数据（如动态题型、灵活校验规则）**上的绝对优势。

## ✨ 核心特性实现
1. **动态无模式设计 (Schema-free)**：摒弃了关系型数据库繁琐的多表联查，将题型、选项、上下限规则全部**动态内嵌 (Embedding)** 存储在单份问卷文档中。
2. **自定义跳转逻辑**：支持极简规则注入（如 `选项A->q3`），前端实时解析并动态隐藏沿途题目。
3. **后端防刷与边界校验**：数据校验规则从数据库动态读取，在后端严格执行多选数量限制和数字大小边界验证。
4. **MongoDB 聚合统计**：利用 `$unwind` 管道聚合操作，结合 Python 字典二次遍历，完美实现多选题选项独立计票与数字题动态求平均值，强制 UTF-8 JSON 输出，无乱码。

## 🛠️ 技术栈
* **数据库**：MongoDB (PyMongo)
* **后端**：Python 3.11 + Flask
* **密码安全**：Bcrypt 加密哈希
* **前端**：原生 HTML5 + CSS3 + JavaScript (Fetch API)
